智能空地协同消防系统——2023年电赛实战总结

2026-03-04
项目实战

1. 项目概况

在 2023 年全国大学生电子设计竞赛中，针对复杂环境下救援人员难以抵达的痛点，我们设计并实现了一套智能空地协同消防系统。该系统由一台智能自主无人机和一台地面消防车组成，通过视觉识别与无线通讯实现高效联动。

所获荣誉：2023年全国大学生电子设计大赛 山东省赛区本科组二等奖。

2. 核心硬件架构

本系统利用了多种嵌入式开发工具和硬件模块进行协同：

  • 主控芯片：采用 STM32 系列单片机，负责底层逻辑控制与运动补偿。
  • 视觉大脑：搭载 OpenMV 摄像头，利用其集成的图像处理算法进行火情实时检测。
  • 硬件设计：利用 Altium Designer 完成了系统核心 PCB 板的绘制与电路打样。
  • 调试工具：开发过程中重度使用 Keil、VS Code 以及串口调试助手进行逻辑优化。

3. 工作流程与逻辑实现

3.1 发现火情（无人机端）

无人机在高空进行巡检，通过 OpenMV 捕捉特定的颜色阈值（模拟火源）。一旦锁定目标，系统会计算出火源相对于无人机的方位信息。

3.2 协同通讯

无人机通过无线通讯模块将火情坐标实时传输至地面消防车。

3.3 快速抵达（消防车端）

地面消防车接收到指令后，结合自身的传感器数据进行路径规划，以最快速度避障并到达模拟现场执行救援任务。

4. 核心代码片段 (示例)

以下是基于 OpenMV 的火情识别逻辑简述：

import sensor, image, time

# 设置颜色识别阈值（模拟火焰色）
fire_threshold = (30, 100, 15, 127, 15, 127)

sensor.reset()
sensor.set_pixformat(sensor.RGB565)
sensor.set_framesize(sensor.QVGA)
sensor.skip_frames(time = 2000)

while(True):
    img = sensor.snapshot()
    blobs = img.find_blobs([fire_threshold])
    if blobs:
        for b in blobs:
            img.draw_rectangle(b.rect())
            # 发送坐标给 STM32 进行无线传输
            print("Fire detected at:", b.cx(), b.cy())
