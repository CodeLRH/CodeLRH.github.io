在正式开始之前，你需要安装两个核心环境：

  • Node.js：Hexo 框架运行的基础。
    • 官方下载地址 (建议选择 LTS 长期支持版)
  • Git：用于将本地代码推送到 GitHub。
    • 官方下载地址

安装完成后，在终端（Git Bash）输入 node -v 和 git --version 检查是否安装成功。

  • 前往 GitHub 配置：

    1. 登录 GitHub，点击右上角头像 -> Settings。
    2. 在左侧菜单找到 SSH and GPG keys。
    3. 点击绿色的 New SSH key 按钮。
    4. Title 随便填（例如填 My Laptop），在 Key 的文本框内粘贴你刚才复制的公钥内容。
    5. 点击 Add SSH key 保存。
